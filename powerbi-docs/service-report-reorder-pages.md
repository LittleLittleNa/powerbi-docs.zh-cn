# <a name="reorder-pages-in-a-report-in-power-bi"></a>在 Power BI 中对报表中的页面重新排序
在 Power BI 中，报表由一个或多个页面组成。  每个页面的底部都有一个标签。  要对报表重新排列，只需选择标签并将其拖动到新位置即可。
